INFORMACION DE GIT

Este parrafo contiene la informacion importante, ademas de un concepto general de lo que es la herramienta para el control de versiones GIT

Git es un sistema de control de versiones que nos ayuda a llevar el historial completo de modificaciones de un proyecto, conociendo en detalle los cambios y quien los hizo
